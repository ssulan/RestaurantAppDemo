# RestaurantAppDemo



This project is a restaurant management system built with ASP.NET Core MVC.

It includes features like:

* Product management
* Category system
* Admin panel
* Customer orders
* Daily,Weekly,Monthly reports.(Excell or Pdf)



Technologies

* ASP.NET Core MVC
* Entity Framework Core
* SQL Server
* HTML, CSS, JavaScript
